Φішечка

Алгебраїчна чиста функціональна мова програмування, яка реалізує синтаксис і семантику PureScript, написана на Elixir для телекомунікаційної платформи Erlang/OTP.

module Main where import Prelude import Effect.Console (log) main :: Effect Unit main = do log "Hello, Φ Phi!" log "Strongly-typed • Concurrent • Distributed"

Why Phi?

BEAM Power + Haskell Beauty

Write pure functional code. Compile to highly scalable, fault-tolerant Erlang/OTP bytecode.

Full Type Safety

Compile-time type checking and inference. Algebraic data types, pattern matching, type classes.

Built-in Concurrency

Actors, OTP behaviours, supervision trees — all with Haskell syntax.

Get Started

git clone git@github.com:synrc/phi
cd phi
